# Radio Commands

>- `radioplay`

- **DESCRIPTION**: Plays/begins the lofi radioplay

<br>

>- `radiovolume`

- **DESCRIPTION**: Changes the volume of the radio

>- `radiostop`
- **DESCRIPTION**: Stops the lofi radio.
